This application provides personalized weather and mood-based music recommendations using the OpenWeather API.

## Features
- Fetch current weather data for any city.
- Suggest music based on weather conditions and mood.
- Responsive and interactive UI.

## Setup
1. Clone the repository:
git clone https://github.com/your-username/weather-music-app.git

2. Open `index.html` in your browser.

## Usage
1. When prompted, enter your OpenWeather API key.
2. Search for a city to view weather details and get music recommendations.

## Technologies
- HTML, CSS, JavaScript
- Bootstrap
